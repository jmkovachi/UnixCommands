## Instructions

To clean my code (i.e., remove any compiled files), type:

```
$ make clean
```

To compile my code, type:

```
$ make
```

To run my code (after compiling it), type:

	For cal
	```
	$ ./cal [[month] year]
	```
	
	For date
	```
	$ ./date [+format]
	```
	
	For head
	```
	$ ./head [-n number] [file...]
	```
	
	For ln
	```
	$ ./ln [-s] target-file link-file
	```
	
	For ls
	```
	$ ./ls [-al] [file...]
	```
	
	For mkdir
	```
	$ ./mkdir [-p] [-m mode] dir...
	```
	
	For env
	```
	$ ./env 
	```
	
	For tail
	```
	$ ./tail [-f] [-c number | -n number] [file]
	```
	
	For true
	```
	$ ./true
	```
	
	For false
	```
	$ ./false 
	```

	For wc
	```
	$ ./wc [-c | -m] [-lw] [file...]
	```
